# Saiyaara 📸🔒

**Saiyaara** is a secure and private photo vault application designed to store personal photos and videos safely.
The app focuses on **privacy, encryption, and simple user experience** so users can keep their memories protected.

---

## ✨ Features

* 🔐 **Secure PIN / Biometric Lock**
* 🖼 **Private Photo & Video Storage**
* 📁 **Album Management**
* ⭐ **Favorite Photos**
* 📤 **Import From Device Gallery**
* ☁ **Optional Cloud Backup**
* 🕵️ **Intruder Selfie Detection**
* 🎭 **Fake Vault Mode**
* 📵 **Screenshot Protection**

---

## 📱 App Screens

* Splash Screen

## 🚀 Getting Started

### 1️⃣ Clone the repository

```

```

### 2️⃣ Install dependencies

```

```

```

```

or

```
npx react-native run-ios
```
